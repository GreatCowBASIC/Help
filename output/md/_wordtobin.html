WordToBin

Syntax:

  stringvar = WordToBin(bytevar)

Command Availability: Available on all microcontrollers

Explanation:

The WordToBin function creates a string of a ANSI (16-byte) characters. The function converts a number to a string consisting of ones and zeros that represents the binary value.

Example:

    string = WordToBin(1)       ' Returns "0000000000000001"

    string = WordToBin(37654)   ' Returns "1001001100010110"

For more help, see ByteToBin

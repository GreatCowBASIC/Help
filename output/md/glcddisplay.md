#### <span id="glcddisplay"></span>GLCDDisplay

</div>

</div>

</div>

<span class="strong">**Syntax:**</span>

``` screen
    GLCDDisplay Off | On
```

<span class="strong">**Explanation:**</span>

Places the GLCD in sleep mode or enables the GLCD for normal operations.

The options are:

``` screen
    OFF
    ON
```
